# Screenshots

Add your screenshots here before submitting:

## Required Screenshots

1. **analytics.png** - Analytics dashboard with real-time metrics
2. **leaderboard.png** - Provider leaderboard with medals
3. **map.png** - Live transaction map
4. **fraud.png** - Fraud detection dashboard

## Tips

- Use full-screen browser (1920x1080 or higher)
- Make sure glassmorphism effects are visible
- Show real data (26 transactions, 0.282 STX)
- Include wallet connected state
- Capture at least one animation in action

## Tools

- macOS: Cmd+Shift+4 (select area)
- Firefox: Right-click → Take Screenshot
- Chrome DevTools: Cmd+Shift+P → "Capture screenshot"

## Optimization

Compress images before committing:
```bash
# Using ImageOptim or:
brew install pngquant
pngquant --quality=65-80 *.png
```
